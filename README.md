**Deployed here - https://ekaterina-dietolog.life**

- [x] **Custom admin panel**
- [x] **Order management**
- [x] **Creating complicated services / programs by admin**
- [x] **Product reviews**
- [x] **Languages support**
- [x] **Email distribute**
- [x] **Smart navigation**
- [x] **Unique design**
